Python Debugger (PDB)

General

?

list available commands

Stack Location

w

where

shows the current line and tree

l

list

show current location

ll

longlist

shows entire funtion or frame around the current location

Stepping

s

step

execute the line and step into it, otherwise move to next

next

execute the current line and move to next

c

continue

continue to the next breakpoint

until

continue until it progresses to the next line

return

continue until the RETURN

Breakpoints

b

break

list current breakpoints set in pdb

b LINENO

set a breakpoint to line LINENO

b LINENO, CONDITION

set a breakpoint to line LINENO if condition is met

condition BREAKPOINT

show the condition set on the specified breakpoint

b FILENAME:LINENO

set a breakpoint to FILENAME at line LINENO

ignore BREAKPOINT HITCOUNT

the specified breakpoint will hit HITCOUNT times and then be ignored

ignore BREAKPOINT 0

re-activate specified breakpoint

disable BREAKPOINT

disable the specified breakpoint

enable BREAKPOINT

enable the specified breakpoint

clear BREAKPOINT

delete the specified breakpoint
